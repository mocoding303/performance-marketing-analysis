# Meta Ads Performance Analysis

SQL-driven marketing analysis of a 90-day Meta Ads 
e-commerce account using PostgreSQL.

## Business questions answered

| # | Question | Layer |
|---|----------|-------|
| 1 | Which campaign generated the highest ROAS? | Descriptive |
| 2 | Why does retargeting outperform prospecting? | Diagnostic |
| 3 | Which day of the week drives the most purchases? | Descriptive |
| 4 | Which placement delivers the lowest CPA? | Descriptive |
| 5 | What is the revenue impact of budget reallocation? | Prescriptive |

## Key finding

Reallocating €1,000 from Prospecting (ROAS 1.41×) to 
Retargeting (ROAS 6.19×) generates a net revenue gain 
of €4,738 — a 44% return on the reallocation decision 
itself, without increasing total budget.

## Tools
PostgreSQL 17 · pgAdmin 4 · Meta Ads Manager · Looker Studio
